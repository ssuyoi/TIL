# 2025-12-30: Java 조건문 (Conditionals) 학습

- 📚 **강의**: 김영한의 자바 입문 섹션 5

오늘은 Java의 조건문인 `if`, `switch` 문과 삼항 연산자에 대해 학습했다. 다양한 예제를 통해 조건에 따른 흐름 제어를 실습했다.

## 1. if 문 (If)

조건에 따라 코드를 실행할지 말지 결정한다.
- `if`: 조건이 참이면 실행.
- `else`: `if` 조건이 거짓이면 실행.
- `else if`: 앞선 `if` 조건이 거짓일 때 새로운 조건을 검사.

**주의사항**:
- 서로 독립적인 조건이라면 여러 개의 `if` 문을 사용해야 한다.
- 서로 연관된 조건(하나만 만족해야 하는 경우)이라면 `if - else if - else` 구조를 사용해야 효율적이다.

```java
int age = 15;

if (age <= 7) {
    System.out.println("미취학");
} else if (age <= 13) {
    System.out.println("초등학생");
} else if (age <= 16) {
    System.out.println("중학생"); // 실행됨
} else {
    System.out.println("성인");
}
```

## 2. switch 문 (Switch)

값이 같은지(`==`) 비교하는 경우 `if` 문보다 가독성이 좋다.

### 기존 switch
`case` 조건에 맞으면 실행되고, `break`가 없으면 다음 `case`로 넘어간다(fall-through).

```java
switch (grade) {
    case 1:
        coupon = 1000;
        break;
    case 2:
        coupon = 2000;
        break;
    default:
        coupon = 500;
}
```

### 새로운 switch (Java 14+)
`->` 를 사용하며, 값을 반환할 수 있어 코드가 훨씬 간결해진다.

`break`가 필요 없다.

```java
int coupon = switch (grade) {
    case 1 -> 1000;
    case 2 -> 2000;
    case 3 -> 3000;
    default -> 500;
};
```

## 3. 삼항 연산자

단순히 참/거짓에 따라 값을 선택해야 할 때 유용하다. `if-else`를 간결하게 대체할 수 있다.
- 구조: `(조건) ? 참_표현식 : 거짓_표현식`

```java
int age = 18;
String status = (age >= 18) ? "성인" : "미성년자";
```

## 4. 실습 문제

### 학점 계산하기 (`ScoreEx`)
점수에 따라 등급(A, B, C, D, F)을 출력하는 문제. `if-else if` 구조를 사용했다.
```java
if (score >= 90) System.out.println("A");
else if (score >= 80) System.out.println("B");
// ...
else System.out.println("F");
```

### 환율 계산하기 (`DollarEx`)
달러가 0 미만, 0, 0 초과일 때 각각 다르게 처리하는 문제. 예외 처리를 먼저 하는 것이 좋다.
```java
if (dollar < 0) {
    System.out.println("잘못된 금액입니다.");
} else if (dollar == 0) {
    System.out.println("환전할 금액이 없습니다.");
} else {
    int won = dollar * 1300;
    System.out.println("환전 금액은 " + won + "원입니다.");
}
```

### 학점에 따른 칭찬 (`GradeSwitchEx`)
학점(String)에 따라 다른 메시지를 출력하는 문제. `switch` 문을 활용했다.
```java
switch (grade) {
    case "A":
        System.out.println("탁월한 성과입니다!");
        break;
    // ...
    default:
        System.out.println("잘못된 학점입니다.");
}
```

### 더 큰 숫자 찾기 (`CondOpEx`)
두 숫자 중 더 큰 숫자를 삼항 연산자를 사용하여 구하는 문제.
```java
int a = 10, b = 20;
int max = (a > b) ? a : b;
System.out.println("더 큰 숫자는 " + max + "입니다.");
```
