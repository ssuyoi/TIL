# 2026-01-02: Java 스캐너(Scanner) 학습

## 1. Scanner 기본

`Scanner` 클래스를 사용하면 사용자로부터 입력을 받을 수 있다. `System.in`을 통해 키보드 입력을 읽어온다.

### 사용법
- `new Scanner(System.in)`: 스캐너 객체 생성.
- `nextLine()`: 엔터를 칠 때까지의 문자열을 읽는다.
- `nextInt()`: 정수를 읽는다.
- `nextDouble()`: 실수를 읽는다.

**주의**: 입력 타입이 맞지 않으면 오류가 발생하므로 주의해야 한다.

## 2. Scanner와 반복문

사용자 입력을 지속적으로 받기 위해 반복문(`while`)과 함께 사용하는 경우가 많다. 특정 키워드(예: "exit")나 숫자(예: 0)가 입력될 때까지 프로그램을 실행하는 구조를 만들 수 있다.

```java
while (true) {
    System.out.print("문자열 입력 (exit 종료): ");
    String str = sc.nextLine();
    if (str.equals("exit")) {
        break;
    }
    System.out.println("입력: " + str);
}
```

## 3. 실습 예제 (Exercises)

### 이름과 나이 입력받기 (`ScannerEx1`)
이름(문자열)과 나이(정수)를 입력받아 출력하는 기본적인 예제.
```java
String name = scanner.nextLine();
int age = scanner.nextInt();
```

### 사이의 정수 출력하기 (`ScannerEx2`)
두 숫자를 입력받아 그 사이의 모든 정수를 출력하는 문제. 두 숫자의 대소 관계를 비교하여(`swap` 로직) 항상 작은 수부터 큰 수까지 출력되도록 구현했다.

### 숫자 입력 후 줄바꿈 처리 (`ScannerWhileEx1`)
`nextInt()`를 사용하여 숫자를 입력받은 후 `nextLine()`을 호출할 때 발생하는 문제를 다뤘다.
- **문제**: `nextInt()`는 숫자만 가져가고 입력 버퍼에 개행문자(`\n`)를 남긴다. 이후 `nextLine()`이 호출되면 남겨진 `\n`을 읽어 빈 문자열을 반환하고 넘어가버린다.
- **해결**: `nextInt()` 호출 직후 `nextLine()`을 한 번 더 호출하여 남아있는 `\n`을 비워준다.

```java
int age = sc.nextInt();
sc.nextLine(); // 버퍼 비우기 (엔터 처리)
String name = sc.nextLine(); // 정상적으로 입력 가능
```
