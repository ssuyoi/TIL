# 2026-01-01: Java 반복문 (Loop) 학습

- 📚 **강의**: 김영한의 자바 입문 섹션 5

오늘은 Java의 반복문인 `while`, `do-while`, `for` 문과 제어 키워드 `break`, `continue`에 대해 학습했다. 반복문의 동작 원리와 효율적인 코드 작성법을 익혔다.

## 1. while 문 (While)

조건에 따라 코드를 반복 실행한다.
- `while` (조건식) { ... }
- 조건식이 참이면 코드 블록을 실행하고, 거짓이면 반복을 종료한다.

### 반복문의 필요성
코드를 복사-붙여넣기하여 반복하는 것은 유지보수에 좋지 않다. `while` 문을 사용하면 간결하게 반복 처리가 가능하다.

```java
// 단순 반복
count = count + 1;
System.out.println("현재 숫자는 " + count);
// ...

// while 문 사용
while (count < 3) {
    count = count + 1;
    System.out.println("현재 숫자는 " + count);
}
```

### 변수 활용과 스코프
반복문 내에서 변수를 적절히 활용하여 합계(sum) 등을 구할 수 있다. 변수 `i`를 사용하여 반복 횟수를 제어하고, `endNum`과 같은 변수로 종료 조건을 유연하게 관리할 수 있다.

```java
int sum = 0;
int i = 1;
int endNum = 3;

while (i <= endNum) {
    sum = sum + i;
    System.out.println("i=" + i + ", sum=" + sum);
    i++;
}
```

## 2. do-while 문

`while` 문과 유사하지만, **조건과 상관없이 무조건 한 번은 코드를 실행**한다는 차이점이 있다.
- `do { ... } while (조건식);`

```java
int i = 10;
do {
    System.out.println("현재 숫자는 " + i); // 무조건 실행됨
    i++;
} while (i < 3); // 조건이 거짓이어도 위 코드는 1회 실행 후 종료
```

## 3. break, continue

반복문의 흐름을 제어하는 키워드이다.
- **`break`**: 반복문을 즉시 종료하고 나간다.
- **`continue`**: 현재 반복 단계의 나머지 코드를 건너뛰고, 다음 반복 조건 검사로 넘어간다.

```java
// continue 예시 (3일 때 건너뛰기)
if (i == 3) {
    i++;
    continue;
}

// break 예시 (합이 10보다 크면 종료)
if (sum > 10) {
    break;
}
```

## 4. for 문

`while` 문보다 구조적으로 더 깔끔하고 직관적이다. 반복에 필요한 **초기식, 조건식, 증감식**을 한 줄에 모아 관리할 수 있다.
- `for (초기식; 조건식; 증감식) { ... }`

```java
// 1부터 10까지 출력
for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}
```

### while vs for
`for` 문은 반복을 위한 변수(카운터)가 `for` 문 내부에 선언되고 관리되므로 스코프가 명확하고 코드가 간결하다. 반면 `while` 문은 변수 선언과 증감식이 분산되어 있어 실수할 가능성이 높다.

## 5. 구조 개선 (Clean Code)

무한 루프 `while(true)`나 `for(;;)`를 사용할 수도 있지만, 반복 횟수가 명확하거나 특정 조건까지 반복해야 한다면 조건식을 명시하는 것이 좋다.
- `for(;;)` 사용: 종료 조건이 루프 내부에 있어 덜 직관적일 수 있음.
- `for(int i = 1; ; i++)`: 반복 변수를 `for` 문 관리 하에 둠. 더 구조적임.

## 6. 중첩 반복문

반복문 안에 또 다른 반복문을 넣을 수 있다.
```java
for (int i = 0; i < 2; i++) {
    System.out.println("외부 for 시작 i = " + i);
    for (int j = 0; j < 3; j++) {
        System.out.println("-> 내부 for " + i + '-' + j);
    }
}
```

## 7. 실습 문제 (Exercises)

### 짝수 출력 (`WhileEx1`, `ForEx1`)
10개의 짝수를 출력하는 문제. `while` 문과 `for` 문 각각으로 구현해보았다.

### 누적 합 계산 (`WhileEx2`, `ForEx2`)
1부터 max까지의 합을 계산하는 문제. `for` 문이 훨씬 간결함을 확인할 수 있었다.

### 구구단 출력 (`NestedEx1`)
중첩 `for` 문을 사용하여 구구단을 출력했다.
```java
for(int i = 1; i <= 9; i++) {
    for(int j = 1; j <= 9; j++) {
        System.out.println( i + " * " + j + " = " + i * j);
    }
}
```

### 피라미드 출력 (`NestedEx2`)
`*`을 피라미드 모양으로 출력하는 문제. 내부 반복문의 조건을 `j <= i`로 설정하여 구현했다.
```java
for(int i = 1; i <= rows; i++) {
    for(int j = 1; j<= i; j++) {
        System.out.print("*");
    }
    System.out.println();
}
```
