# TIL & Lecture Practice

개발 강의를 수강하며 진행한 실습 코드와 학습 내용을 정리하는 공간입니다.

## 📂 폴더 구조

### 1. `lectures/`
- 강의에서 진행한 실습 프로젝트와 예제 코드가 저장됩니다.
- 각 강의 또는 챕터별로 프로젝트를 구분하여 관리합니다.

### 2. `TIL/` (Today I Learned)
- 강의를 들으며 배운 핵심 개념, 이론, 트러블슈팅 경험 등을 정리합니다.
- 주제별 혹은 날짜별로 학습 기록을 남깁니다.
